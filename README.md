# JavaFX E-Commerce Application

A desktop e-commerce application built with **JavaFX** and **MySQL**, using **JDBC** for data access and **Maven** for dependency management.  
Includes user/admin flows such as product browsing, favoriting, placing/processing orders and basic CRUD operations.

## Tech Stack
- **Language:** Java
- **UI:** JavaFX
- **Database:** MySQL
- **Data Access:** JDBC
- **Build:** Maven (with Maven Wrapper)

## Features
- User authentication (login/register)
- Product listing and search/filter
- Mark products as favorite (like)
- Cart and checkout flow
- Admin panel: manage products/orders/users
- MySQL persistence with JDBC
